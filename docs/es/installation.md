# Instalación

## Instalar el paquete

```bash
pip install django-form-alpine
```

## Agregar a INSTALLED_APPS

```python
INSTALLED_APPS = [
    # ...
    "django_form_alpine",
    # ...
]
```

Eso es todo. La librería carga sus scripts automáticamente a través del sistema de media de Django cuando usas alguno de los mixins incluidos.
